# 宠物信息管理系统需求文档

## 1. 项目概述
面向宠物主人的个人宠物信息管理系统，帮助用户系统化管理宠物的全生命周期信息，包括基础档案、健康记录、疫苗提醒、消费追踪等功能。

## 2. 目标用户
宠物主人（个人用户）

## 3. 核心功能需求

### 3.1 用户管理模块
- 用户注册/登录（账号密码、第三方登录可选）
- 用户个人信息管理（头像、昵称、联系方式等）
- 密码找回功能
- 多宠物管理（一个用户可以添加多个宠物）

### 3.2 宠物档案管理模块
- 宠物基础信息录入：
  - 基本信息：姓名、品种、性别、出生日期、体重、毛色、芯片号
  - 特征描述、性格特点
  - 宠物照片上传
- 宠物信息编辑和删除
- 宠物档案详情查看

### 3.3 疫苗提醒模块
- 疫苗记录录入：疫苗名称、接种日期、接种单位、下次接种提醒日期、备注
- 智能提醒：系统在疫苗到期前推送提醒（站内消息、邮件/短信可选）
- 疫苗记录历史查询
- 常见疫苗库预设（狂犬疫苗、三联、五联等）

### 3.4 健康记录模块
- 就诊记录：就诊时间、医院名称、医生诊断、治疗方案、用药情况、费用、附件上传（诊断报告、处方等）
- 体重记录：定期记录体重，生成体重变化趋势图
- 驱虫记录：驱虫类型、日期、下次驱虫提醒
- 过敏记录：过敏原、过敏反应、处理方式
- 手术记录：手术名称、时间、医院、术后注意事项
- 健康数据统计和可视化展示

### 3.5 日常开销追踪模块
- 消费记录分类：食品、医疗、美容、用品、保险、其他
- 消费记录录入：消费时间、金额、商家、备注、附件上传（小票照片）
- 消费统计：月度/季度/年度消费报表，分类占比分析
- 消费趋势图表展示

### 3.6 系统功能
- 数据导出：支持宠物档案、健康记录、消费记录导出为PDF/Excel
- 数据备份与恢复
- 深色/浅色主题切换
- 多语言支持（可选）

## 4. 非功能需求
- 响应式设计，支持PC端和移动端访问
- 数据安全性：用户数据加密存储，接口权限控制
- 系统可用性：界面简洁易用，操作流畅
- 性能要求：页面加载时间<2秒，接口响应时间<500ms
- 可扩展性：支持后续功能扩展（如社区交流、宠物匹配等）

## 5. 边界说明
- 本系统为个人使用，暂不支持多用户共享宠物信息
- 暂不包含在线购物、在线问诊等商业功能
- 提醒功能初期实现站内提醒，后续可扩展短信/邮件提醒
